===========================================================================================================
                                         Binary Search
===========================================================================================================

Binary search is a search algorithm that finds the position of a target value within a sorted collection.
Instead of checking each element one by one, it compares the target with the middle element and eliminates
half of the remaining elements with each comparison.

in simple words,
    Binary Search is an efficient searching algorithm used to find an element in a sorted array or list.
    Instead of checking each element (like Linear Search), it:
        Divides the search space into half each time
        Eliminates half of the elements in every step

Time Complexity:
    Best: O(1)
    Average/Worst: O(log n)

How Binary Search Works:
    Find the middle element
    Compare it with target:
        If equal → found
        If target < mid → search left half
        If target > mid → search right half
    Repeat until found or range ends

When to Use Binary Search:
    ✅ Data is sorted (very important)
    ✅ Random access is possible (arrays, not linked lists)
    ✅ Need faster search than O(n)
